The Problem

Why Does Plant Care Fail?

Urban gardeners face persistent challenges maintaining healthy plants — especially when away from home.

🏜️

Under-watering

Plants dry out completely during extended absences, leading to irreversible damage and poor growth.

🌊

Over-watering

Excessive moisture causes root rot, fungal infections, and nutrient leaching from the soil.

📵

Zero Visibility

No way to monitor plant health remotely, creating anxiety when away from home for days.

How The System Works

Follow the journey from dry soil to a perfectly watered plant — fully automated, remotely monitored.

💧

Sensing

Soil Moisture Detected

A capacitive soil moisture sensor is inserted into the pot. It measures moisture by detecting changes in electrical capacitance — no metal corrosion, long lifespan.

GPIO Pin 34 · ADC1_CH6
2
📡
🖥️
Processing

ESP32 Reads & Converts Data

The ESP32 reads 12-bit ADC values (0–4095) and converts them to a moisture percentage using a calibrated linear mapping function.

map(val, 1407, 4095, 100, 0)
3
📲
📊
Cloud & App

Live Data Sent to Blynk App

Moisture data streams to the Blynk cloud platform via WiFi. A real-time gauge widget displays current moisture with colour-coded alerts — Red <30%, Yellow 30–60%, Green >60%.

2.4GHz WiFi · <1sec latency
4
⚡
🔌
Decision Logic

Threshold Check: Below 30%?

If moisture drops below 30%, the system automatically triggers the relay module on GPIO Pin 25, which switches the pump ON. Electrical isolation protects the ESP32 from the high-current pump circuit.

5V Relay · 10A max load · flyback diode
5
💦
🚰
Actuation

Pump Waters the Plant

A 12V DC submersible pump delivers 2–5 litres per minute of water. Watering continues until moisture exceeds the threshold, then the relay cuts power to the pump automatically.

12V DC · 2–5 L/min flow rate
6
🌿
✅
Manual Override

Remote Control Anytime

A push-button in the Blynk app lets you manually trigger the pump from anywhere in the world — ideal for fertiliser application or overriding the automation when needed.

Cloud control · unlimited range
Key Features

What Makes It Smart

The system combines precision hardware with intuitive IoT connectivity for a truly hands-free experience.

🤖

Autonomous Mode

Threshold-based (30%) automatic watering — no human input required. The system monitors 24/7 and reacts in real time.

📱

Manual Override

One-tap pump control from the Blynk app anywhere in the world via cloud connectivity — unlimited range.

📈

Real-time Monitoring

Live moisture gauge with colour-coded indicators streamed to your phone with under 1 second response latency.

🛡️

Corrosion-resistant Sensor

Capacitive sensor technology — no metal probes, no corrosion, far superior lifespan vs. resistive sensors.

💧

Water Efficiency

Reduced water usage by 35% compared to traditional scheduled watering methods in 30-day testing.

🔧

Modular Architecture

Designed to scale — multi-zone expansion, solar power, and Weather API integration are future-ready.
